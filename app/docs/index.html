Formularios API (1.0.0)

Download OpenAPI specification:

API para gestión de formularios de registro con campos dinámicos.

Características

  • Registro de formularios con validación de campos obligatorios
  • Soporte para campos adicionales dinámicos
  • Validación de documentos únicos (tipo + número)
  • Persistencia en DynamoDB

Formularios

Operaciones de gestión de formularios

Listar formularios

Obtiene todos los formularios registrados en el sistema.

Los campos adicionales dinámicos se devuelven en el nivel superior de cada formulario.

Registrar formulario

Registra un nuevo formulario con campos obligatorios y opcionales.

Campos obligatorios

  • nombres, apellidos, tipoDocumento, numeroDocumento, email, celular, tratamientoDatos

Campos adicionales

Se pueden enviar campos adicionales dinámicos (ej: empresa, cargo, ciudad). Estos campos se guardan y devuelven en el nivel superior del JSON.

Validaciones

  • El documento (tipo + número) debe ser único
  • El tratamiento de datos debe ser aceptado (true)

required

Correo electrónico válido

celular
required
string^\+?[0-9]{9,15}$

Número de celular (9-15 dígitos, puede incluir +)

required
boolean

Aceptación del tratamiento de datos personales (debe ser true)
